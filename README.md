# Project Name: 


__Indoor Plant Care & Store__

Site Link: https://a-9-green-nest.netlify.app/

GreenNest is an elegant single-page web application built for plant lovers who want to nurture and decorate their homes with healthy indoor plants. The platform allows users to explore plant care guides, buy plants, and book expert consultations ‚Äî ensuring a greener and healthier living space during any season.

# Key Features: 

* 
* A responsive, visually appealing plant care and store platform.

* Implement secure Firebase authentication (Signup, Login, Google Sign-In, Forgot Password).

* Data Fetched from JSON for product/service listings.

* Protected route for service details and profile management.

* Maintained SPA functionality with a minimalist and calming design.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/jira-clone.git
   cd jira-clone
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to start using the app.
